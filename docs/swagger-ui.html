🏦 LLM-Lite Underwriter API

파라메트릭 보험 자동 인수심사 시스템 API 문서

📚 API 개요

이 API는 자연어 입력을 통해 파라메트릭 보험상품을 자동으로 설계하는 인수심사 시스템입니다. 고객이 한국어로 보험 요구사항을 입력하면, AI 에이전트가 위험 분석, 가격 계산, 보험상품 설계를 수행합니다.

  • 비동기 처리: 백그라운드에서 인수심사 진행
  • 실시간 진행률: 처리 단계별 상태 확인
  • 다양한 알림: Webhook, 파일, 메시지큐 지원
  • 배치 처리: 여러 요청 일괄 처리

POST/underwrite

인수심사 요청을 제출합니다. 자연어 입력을 받아 비동기로 보험상품을 설계합니다.

curl -X POST "http://localhost:8000/api/v1/underwrite" \ -H "Content-Type: application/json" \ -d '{"request_id":"test_001","user_input":"태풍 보험상품 설계","callback":{"type":"webhook","webhook":{"url":"https://example.com/callback"}}}'

GET/tasks/{task_id}

작업 상태를 조회합니다. 진행률, 결과, 에러 정보를 확인할 수 있습니다.

curl "http://localhost:8000/api/v1/tasks/underwriter_20250121_143052_a1b2c3d4"

POST/notifications/test-webhook

Webhook 연결을 테스트합니다. 실제 결과 전송 전에 연결 상태를 확인할 수 있습니다.

curl -X POST "http://localhost:8000/api/v1/notifications/test-webhook" \ -H "Content-Type: application/json" \ -d '{"url":"https://httpbin.org/post","headers":{"Content-Type":"application/json"}}'

GET/health

시스템 상태를 확인합니다. 서버와 의존성 시스템의 정상 동작 여부를 체크합니다.

curl "http://localhost:8000/api/v1/health"
